# all-types-api-example

REST API з прикладами роботи з різними типами файлів: JSON, Excel, відео, архіви, зображення.

## Вимоги

- **Node.js** >= 20.0.0
- **npm**

## Встановлення

```bash
npm install
```

## Запуск сервера

```bash
# Production
npm start
```

```bash
# Development (з авто-перезапуском при змінах)
npm run dev
```

Сервер запускається на `http://localhost:3000`.

Swagger-документація доступна за адресою `http://localhost:3000/api-docs`.

> Playwright автоматично стартує сервер перед тестами, якщо він ще не запущений.
